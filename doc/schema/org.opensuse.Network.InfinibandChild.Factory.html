string newDevice(string name, dict config)

Argument config is mapped to element <infiniband:child> of the device document. It is a dict-based type that provides the following members:

  • mode (builtin-infiniband-mode). Infiniband IPoIB connection mode
  • multicast (builtin-infiniband-umcast). Infiniband IPoIB user multicasts policy
  • device (string). This is a string
  • pkey (uint16). Infiniband partition key

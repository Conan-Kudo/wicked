Type port_t

This is a uint32 enumerated type, which can take the following values:

  • default (0)
  • tp (1)
  • aui (2)
  • bnc (3)
  • mii (4)
  • fibre (5)

Type wol-flag-mask

This is a uint32 bitmap type, which is a combination of zero or more of the following flags:

  • phy (0)
  • unicast (1)
  • multicast (2)
  • broadcast (3)
  • arp (4)
  • magic (5)
  • secure-on (6)

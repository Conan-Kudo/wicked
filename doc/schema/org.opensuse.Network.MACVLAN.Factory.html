string newDevice(string name, dict config)

Argument config is mapped to element <macvlan> of the device document. It is a dict-based type that provides the following members:

  • device (string). This is a string
  • address (ethernet-address).
  • mode (builtin-macvlan-mode).
  • flags (builtin-macvlan-flag-bits).
